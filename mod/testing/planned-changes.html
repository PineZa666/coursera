Summary of Observations

  • Some confusion around activity categories vs titles.
  • Desire for edit/delete to correct mistakes.

Changes & Justifications

  • Reorder fields to category → title → minutes — matches user mental model observed.
  • Add edit/delete on Activities page — users expect to fix entries.
  • Enhance accessibility (larger tap targets, focus states) — improves mobile usability.
  • Add micro‑tour on Info page — helps first‑time users discover core flow.
